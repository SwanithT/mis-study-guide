Chapter 3: Platforms & Network Effects

Understanding how platforms grow, dominate markets, and create value.

---

Vocabulary

Network Effects: When a product becomes more valuable as more people use it.
Example: Instagram becomes more useful as more friends join.
Metcalfe’s Law: The value of a network grows roughly with the square of the number of users.
Example: Doubling users more than doubles value.
Platform: A product or service that allows others to build complementary products.
Example: App Store allows developers to create apps.
Complementary Products: Products that add value to another product.
Example: Apps increase value of smartphones.
One-Sided Market: A market that gets value from one group of users.
Example: Messaging apps like WhatsApp.
Two-Sided Market: A market connecting two different user groups.
Example: Uber connects drivers and riders.
Cross-Side Network Effects: Growth on one side increases value for the other side.
Example: More riders → more drivers join Uber.
Same-Side Network Effects: Value increases within the same group.
Example: More friends on Snapchat increases value.
Bundling: Selling multiple products together as one package.
Example: Microsoft Office bundle.
Switching Costs: The cost of changing from one system to another.
Example: Moving files and learning a new OS.
Marginal Cost: The cost of producing one additional unit.
Example: Software has near-zero marginal cost.
Winner-Take-All Market: A market where one firm dominates due to network effects.
Example: Google in search.
Convergence: When separate markets combine into one.
Example: Phones + cameras + GPS → smartphones.
Envelopment: When one product absorbs another product’s features.
Example: Facebook adding messaging.
Freemium Model: A free version with paid upgrades.
Example: Zoom free vs premium plans.
